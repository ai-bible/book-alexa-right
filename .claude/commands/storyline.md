name: storyline
---

Manage character storylines. Usage:

**Creation & Status:**
- `/storyline status [character-name]` - Show current storyline state
- `/storyline new [character-name]` - Create new storyline (interactive)

**Development:**
- `/storyline develop [character-name]` - Collaborative storyline development

**Updates:**
- `/storyline update [character-name] [event-description]` - Update based on event

**Analysis:**
- `/storyline connections [character-name]` - Show storyline connections

Invokes the storyline-developer agent.
