# Финальная зачистка упоминаний хрономатериала

**Дата**: 2025-10-13
**Принцип**: Хрономатериал — это **специальная технология**, а не бытовая вещь. Упоминать только при **критической сюжетной необходимости**.

---

## Проблема

Хрономатериал упоминался **слишком часто** в контексте:
- "Атмосферы" (гудение в Башне Книжников)
- "Сенсорики" (запах в медотсеке)
- Описания обычных предметов (одеяло, одежда, кушетка, лифт)

**Эффект**: Worldbuilding становился навязчивым. Читатель устаёт от "Look at my cool tech!". Фокус смещался с людей на технологию.

---

## Принцип "Show, Don't Tell Tech"

### ❌ Плохо (удалено):
> "Башня Книжников встречала приглушённым гудением **хрономатериалов** — звук был таким низким, что скорее ощущался костями, чем ушами."

**Почему плохо**:
- Гудение не важно для сюжета
- Можно просто "Башня гудела" — эффект тот же
- Упоминание хрономатериала = лишняя информация

### ✅ Хорошо (оставлено в строке 33 сцены 2.1):
> "Воздух здесь был густым, словно мёд. Даже её ускоренный метаболизм замедлился."

**Почему хорошо**:
- Показано **через ощущение** персонажа
- Не объясняем "почему" (хрономатериалы)
- Читатель понимает: богатые живут в другом времени **буквально**

## Выполненные зачистки

### Удалено (финальный раунд):

#### 1. **Башня Книжников - гудение** (сцена 1.2, строка 3)
```
БЫЛО: Башня Книжников встречала приглушённым гудением хрономатериалов.
СТАЛО: Башня Книжников встречала приглушённым гудением.
```
**Обоснование**: Источник звука не важен. Гудение создаёт атмосферу, материал — нет.

#### 2. **Коридор - холод** (сцена 1.2, строка 11)
```
БЫЛО: Холодно. Всегда холодно на шестьдесят третьем. Алекса не помнила,
      чтобы хрономатериалы грели. Они просто были. Как лёд, который не тает.
СТАЛО: Холодно. Всегда холодно на шестьдесят третьем.
```
**Обоснование**: Холод — это ощущение. Не нужно объяснять причину через технологию.

#### 3. **Медотсек - запах** (сцена 1.3, строка 11)
```
БЫЛО: Коридор медицинского отсека пах антисептиком и хрономатериалами.
СТАЛО: Коридор медицинского отсека пах антисептиком.
```
**Обоснование**: Запах антисептика = медицина. Запах хрономатериалов = worldbuilding ради worldbuilding.

#### 4. **Золотая Терраса - описание** (сцена 2.1, строка 15)
```
БЫЛО: "Золотая Терраса" — один из самых элитных районов Верхнего Города,
      где воздух был насыщен хрономатериалами, а время текло медленнее обычного.
СТАЛО: "Золотая Терраса" — один из самых элитных районов Верхнего Города.
```
**Обоснование**:
- Эффект показан позже (строка 33) через ощущение Алексы
- Объяснять дважды = избыточность
- Читатель увидит эффект сам, когда Алекса почувствует

## Правило для будущего

### Когда можно упомянуть хрономатериал:

1. ✅ **Критический сюжетный момент**
   - Пример: персонаж впервые видит хрономатериал, это важно для понимания мира
   - Пример: технология ломается, это влияет на сюжет

2. ✅ **Показ социального неравенства**
   - Пример: богатые живут в "пузыре замедленного времени", бедные — нет
   - НО: показывать через **ощущения персонажа**, не объяснения

3. ✅ **Объяснение механики действия**
   - Пример: персонаж впервые использует технологию, нужно понять "как это работает"
   - НО: только если это влияет на **следующее действие**

### Когда НЕЛЬЗЯ упоминать:

1. ❌ **"Атмосфера"**
   - Гудение, запах, цвет — не упоминать материал
   - Просто "гудело", "пахло озоном", "светилось"

2. ❌ **Бытовые предметы**
   - Одеяло, одежда, кушетка, лифт — не упоминать материал
   - Читателю всё равно, из чего сделан лифт

3. ❌ **Повтор в одной сцене**
   - Если уже упомянули = больше не упоминать
   - Эффект показан один раз = достаточно

## Философия подхода

### Книга о людях, не о технологии

> "Мы пишем про людей, хрономатериал это особая технология и просто так вставлять такие вещи не нужно и даже вредно. Это не что-то обычное для нашего мира, а значит должно появляться СТРОГО ПРИ НЕОБХОДИМОСТИ."

**Перевод в практику**:
- Если убрал упоминание хрономатериала, и сцена **не изменилась** = правильно убрал
- Если убрал, и сцена **потеряла смысл** = нужно было оставить

**Тест "А если это пластик?"**:
- Заменяем "хрономатериал" на "пластик"
- Если сцена не изменилась = упоминание лишнее
- Пример: "Одеяло из пластика" vs "Одеяло" = ноль разницы

## Сравнение: "Show Tech" vs "Show Effect"

### ❌ Show Tech (удалено):
> "Воздух был насыщен хрономатериалами, а время текло медленнее обычного."

**Проблема**: Объясняем технологию. Читатель получает **информацию**, но не **ощущение**.

### ✅ Show Effect (оставлено):
> "Воздух был густым, словно мёд. Даже её ускоренный метаболизм замедлился."

**Преимущество**: Показываем **ощущение персонажа**. Читатель чувствует эффект **вместе с Алексой**. Технология скрыта, но присутствует.
