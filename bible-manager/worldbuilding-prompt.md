# Промпт для анализа новых элементов мира и интеграции в канон

## Инструкции для использования

1. Скопируйте текст главы, которую нужно проанализировать
2. Вставьте этот текст в запрос к ИИ вместе с этим промптом
3. При необходимости приложите текущую версию библии мира или таблицу канона для проверки на противоречия

---

## ПРОМПТ ДЛЯ ИИ

Проанализируй следующий текст главы и выдели все новые элементы мира "Нью-Москва", которые появились в этой главе. Твоя задача:

1. Найти и классифицировать все новые элементы мира по категориям
2. Оценить их совместимость с существующим каноном
3. Предложить уровень канона для каждого элемента
4. Определить потенциальные противоречия или вопросы

### КАТЕГОРИИ ЭЛЕМЕНТОВ ДЛЯ АНАЛИЗА:

- **Технологии и материалы**: новые устройства, материалы, научные концепции
- **Социальная структура**: общественные группы, классы, организации, правила поведения
- **Персонажи**: новые действующие лица, их особенности, модификации, мотивации
- **География и локации**: новые места, районы, здания, их особенности
- **История мира**: исторические события, упомянутые в тексте
- **Культура и быт**: особенности повседневной жизни, искусство, развлечения
- **Проекты и разработки**: научные или социальные инициативы

### КРИТЕРИИ ОЦЕНКИ ЭЛЕМЕНТОВ ДЛЯ РАСПРЕДЕЛЕНИЯ ПО УРОВНЯМ КАНОНА:

**Уровень 1 (Основной канон)** - базовые, критически важные для мира и сюжета элементы:
- Создает фундаментальное правило мира
- Имеет потенциал стать центральным для сюжета
- Напрямую связан с главными конфликтами истории
- Невозможно изменить без серьезных последствий для всего повествования

**Уровень 2 (Расширенный канон)** - важные детали, дополняющие основной канон:
- Расширяет понимание элементов первого уровня
- Имеет значение для текущей сюжетной линии
- Упоминается несколько раз и в разных контекстах
- Соответствует логике мира и его правилам

**Уровень 3 (Возможный канон)** - интересные, но пока второстепенные детали:
- Упоминается вскользь или однократно
- Интересная деталь, но не критичная для текущего сюжета
- Требует дополнительной проработки
- Потенциально может быть развита в будущих главах

**Уровень 4 (Неканон)** - проблематичные или противоречивые элементы:
- Противоречит установленным правилам мира
- Создает логические несоответствия
- Требует значительной переработки для интеграции
- Не вписывается в общую концепцию и атмосферу мира

### ИНСТРУКЦИИ ПО ОЦЕНКЕ СОВМЕСТИМОСТИ:

- **Совместимый элемент**: логично вписывается в существующий канон, не создает противоречий
- **Требует проверки**: не очевидно противоречит, но нуждается в дополнительной проверке или уточнении
- **Потенциальное противоречие**: есть заметные несоответствия с ранее установленными элементами
- **Явное противоречие**: напрямую противоречит фундаментальным правилам мира

Важно: для элементов с оценкой "Требует проверки" или хуже, обязательно укажи конкретное противоречие или вопрос.
