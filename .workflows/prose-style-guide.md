# Prose Style Guide для романа об Алексе Райт

## Обзор

Этот guide определяет стилистические подходы для разных типов сцен в романе. Каждый тип сцены требует своего тона, ритма и приёмов для максимального эффекта.

**Ключевой принцип**: Стиль определяется содержанием сцены, а не главой или актом. В одной главе могут быть сцены разных типов.

## Как определить тип сцены

При написании сцены определите её основной фокус:
- Что происходит физически?
- Какова эмоциональная доминанта?
- Что важнее: внешнее действие или внутренний процесс?
- Есть ли насилие/угроза насилия?

Одна сцена может комбинировать несколько типов (например, социальное взаимодействие + саспенс), в этом случае используйте доминирующий тип как основу и добавляйте элементы второстепенных.

## Типы сцен и стили

### 1. Насилие и жестокость

**Референс**: Joe Abercrombie
**Когда применять**: Физические конфликты, пытки, жестокие сцены, драки, убийства

#### Ключевые характеристики:

**Длина предложений**:
- 5-10 слов в среднем
- Фрагменты при экстремальном действии
- Никаких сложных конструкций

**Фокус**:
- Физические детали над эмоциями
- Конкретные действия: "Кулак встретил челюсть" вместо "Он ударил его"
- Последствия насилия показаны буквально

**Голос**:
- Циничный внутренний монолог
- Чёрный юмор в критические моменты
- Отстранённость персонажа от собственных действий

**Ритм**:
- Резкий, рваный
- Паузы через короткие абзацы
- Чередование действия и реакции

#### Примеры техник:

**Хорошо** ✅:
```
Кулак. Челюсть. Хруст.

Алекса качнулась назад. Металлический привкус. Кровь.

Второй удар. Не успела уклониться.
```

**Плохо** ❌:
```
Она почувствовала сильную боль, когда его кулак с ужасающей силой врезался в её лицо, заставив её пошатнуться назад.
```

#### Практические guidelines:

- Избегайте наречий ("сильно", "жестоко", "больно") - показывайте через детали
- Используйте односложные глаголы: hit, break, crack, slam
- Минимум внутреннего монолога во время действия
- Реакции персонажа через физиологию: дрожь, тошнота, адреналин
- Циничные мысли ПОСЛЕ насилия, не во время

### 2. Детектив/Расследование

**Референс**: Gillian Flynn
**Когда применять**: Алекса ищет информацию, анализирует улики, расследует прошлое, подозревает обман

#### Ключевые характеристики:

**Ненадёжный нарратор**:
- Алекса может ошибаться в выводах
- Её восприятие окрашено паранойей
- Читатель должен сомневаться в её интерпретациях

**Тон**:
- Тёмные мысли
- Подозрительность ко всем
- Острые психологические наблюдения о людях
- Cynical insights

**Детали**:
- Фокус на маленьких несоответствиях
- Замечает то, что другие пропускают
- Физические детали как улики (жесты, взгляды, паузы)

**Структура**:
- Наблюдение → интерпретация → сомнение в интерпретации
- Вопросы без ответов
- Loops of paranoid thinking

#### Примеры техник:

**Хорошо** ✅:
```
Дэвид задержал взгляд на экране. Секунда. Две. Слишком долго для случайного взгляда.

Или она параноит? Возможно. Вероятно.

Но его голос изменился. Она это знала. Или думала, что знала.
```

**Плохо** ❌:
```
Алекса поняла, что Дэвид что-то скрывает, потому что он вёл себя странно.
```

#### Практические guidelines:

- Показывайте мыслительный процесс: наблюдение → гипотеза → контр-гипотеза
- Используйте вопросительные предложения в internal monologue
- Детали среды как зацепки (но не очевидные)
- Алекса замечает микровыражения, противоречия, паузы
- Paranoid undertone: "Или она ошибалась?"
- Никаких прямых выводов - всё через сомнение

### 3. Sci-Fi элементы (технологии)

**Референс**: William Gibson (Soft Sci-Fi)
**Когда применять**: Работа с редактированием памяти, временные технологии, консоли, временные браслеты, технический world-building

#### Ключевые характеристики:

**Технологии обыденны**:
- Для персонажей это routine, не чудо
- Показ через использование, не объяснение
- Как мы описываем использование смартфона - просто и буднично

**Специфичность без exposition**:
- Конкретные детали: "Зелёный индикатор. Доступ разрешён."
- НЕТ объяснений: "как работает мнемоническое редактирование"
- Термины используются естественно, контекст даёт понимание

**Киберпанк-эстетика**:
- Холодные материалы: металл, стекло, пластик
- Искусственное освещение
- Технологии как продолжение тела

**Показ через действие**:
- Персонаж взаимодействует с tech
- Результаты видны немедленно
- Никаких info dumps о принципах работы

#### Примеры техник:

**Хорошо** ✅:
```
Консоль откликнулась. Мнемонический паттерн развернулся на экране — сеть связей, мерцающая синим. Алекса коснулась узла. Увеличение. Воспоминание: запах кофе, голос матери, тёплое утро.

Редактирование. Удалить голос. Оставить запах.

Паттерн пересобрался. Чисто.
```

**Плохо** ❌:
```
Алекса использовала передовую технологию мнемонического редактирования, которая позволяла изменять воспоминания на нейронном уровне через квантовую интерференцию синаптических связей.
```

#### Практические guidelines:

- Одним предложением описывайте tech interaction
- Результат важнее процесса
- Терминология без курсива и объяснений (читатель поймёт из контекста)
- Сенсорные детали tech: звуки (жужжание, писк), тактильные (холод, вибрация), визуальные (цвет индикаторов)
- Технологии влияют на людей - показывайте социальные/психологические эффекты
- Временные браслеты: просто цифры и цвет, без drama ("6 лет 4 месяца 18 дней. Красный.")

### 4. Внутренняя рефлексия

**Подход**: Минимальная
**Когда применять**: Размышления о памяти, identity, философские моменты

#### Ключевые характеристики:

**Минимум внутреннего монолога**:
- Больше действий и наблюдений
- Краткие вспышки мыслей, не длинные рассуждения
- Эмоции через физические реакции

**Показ через действие**:
- Вместо "Она думала о..." → показать, как это влияет на поведение
- Вместо абстрактных размышлений → конкретный триггер и физическая реакция

**Краткость**:
- Одно-два предложения max для философской мысли
- Быстрый переход к действию/наблюдению
- Избегать длинных passages introspection

#### Примеры техник:

**Хорошо** ✅:
```
Какие воспоминания настоящие? Алекса сжала кулаки. Не важно сейчас.

Кот мурлыкал. Тёплый. Реальный. Этого достаточно.
```

**Плохо** ❌:
```
Алекса долго размышляла о природе памяти и личности. Если её воспоминания были изменены, кто она на самом деле? Возможно, личность - это не сумма воспоминаний, а что-то более глубокое, недоступное редактированию. Или наоборот, мы лишь конструкты наших воспоминаний, и изменив их, можно изменить всё...
```

#### Практические guidelines:

- Философские вопросы формулируйте кратко (одно предложение)
- Сразу уходите от абстракции к конкретике
- Emotional state через сенсорику: "Холод в груди" вместо "Она чувствовала пустоту"
- Если нужен момент рефлексии - показывайте его через взаимодействие с объектом (кот, фото, браслет)
- Алекса прагматична - быстро обрывает философские мысли и возвращается к действию

### 5. Саспенс/Напряжение

**Подход**: Thriller-стиль
**Когда применять**: Преследование, скрытность, побег, угроза раскрытия, deadline pressure

#### Ключевые характеристики:

**Быстрый темп**:
- Короткие параграфы (1-3 предложения)
- Быстрая смена фокуса
- Никаких отступлений

**Нарастающее напряжение**:
- Сенсорные детали усиливают тревогу
- Каждый абзац добавляет новый элемент угрозы
- Countdown или временное давление явно или неявно

**Клиффхэнгеры**:
- Заканчивайте beats на моменте неопределённости
- Вопросы без ответов
- Interrupted actions

**Физиология стресса**:
- Учащённое сердцебиение
- Потные ладони
- Сухость во рту
- Острое восприятие звуков

#### Примеры техник:

**Хорошо** ✅:
```
Шаги в коридоре. Приближаются.

Алекса замерла. Экран всё ещё светился. Закрыть окно? Или подозрительно?

Дверь начала открываться.

Три секунды. Два.

Слишком поздно.
```

**Плохо** ❌:
```
Алекса услышала шаги и поняла, что кто-то идёт. Она нервничала, потому что на экране была конфиденциальная информация, и она не знала, успеет ли закрыть окно до того, как человек войдёт.
```

#### Практические guidelines:

- Используйте короткие абзацы для acceleration
- Countdown: "Три секунды. Два. Один."
- Sensory details острые и конкретные
- Неполные предложения для передачи паники
- Показывайте multiple threats accumulating
- Каждое предложение должно усиливать tension
- Delay resolution - растягивайте момент before discovery/confrontation

### 6. Социальное взаимодействие

**Референс**: Элинор Катон
**Когда применять**: Офисные разговоры, встречи с коллегами, формальное общение, классовые различия

#### Ключевые характеристики:

**Формальность с подводными течениями**:
- Вежливая поверхность
- Напряжение в невысказанном
- Социальные ритуалы и их нарушения

**Классовое напряжение**:
- Тонкие маркеры статуса (речь, жесты, одежда)
- Power dynamics в диалоге
- Кто контролирует разговор

**Подтекст важнее текста**:
- То, что НЕ сказано, важнее слов
- Паузы, недоговорённости
- Социальные маски

**Детальное наблюдение**:
- Микровыражения
- Tone shifts
- Body language противоречит словам

#### Примеры техник:

**Хорошо** ✅:
```
— Хорошая работа, Алекса. — Себастьян кивнул. Профессионально. Дистанция в голосе чётко отмерена.

Она заметила. Он раньше называл её по имени иначе. Теплее. Теперь — как сотрудницу.

— Спасибо, господин Грей. — Тоже формально. Играем в эту игру.

Пауза. Он ждал, что она уйдёт. Она не ушла.

Его челюсть напряглась. Едва заметно.
```

**Плохо** ❌:
```
Себастьян разговаривал с Алексой холодно и формально, показывая свой высокий статус. Она чувствовала напряжение между ними.
```

#### Практические guidelines:

- Dialogue с подтекстом: персонажи говорят не то, что думают
- Показывайте power dynamics через: кто прерывает, кто задаёт вопросы, кто заканчивает разговор
- Классовые маркеры subtle: качество одежды, выбор слов, где человек стоит
- Паузы значимы - используйте их
- Противоречие между словами и body language
- Социальные ритуалы (приветствия, формальности) и их нарушения

### 7. Профессиональная работа

**Подход**: Психологический
**Когда применять**: Алекса редактирует память клиентов, профессиональная routine

#### Ключевые характеристики:

**Влияние работы на психику**:
- Как редактирование чужих воспоминаний влияет на Алексу
- Эмоциональная цена профессиональной отстранённости
- Blur между чужими травмами и своими

**Профессиональная маска**:
- Внешнее спокойствие и компетентность
- Внутреннее воздействие
- Механизмы защиты (цинизм, рутинизация)

**Этические undercurrents**:
- Моральные вопросы без прямых answers
- Оправдания, которые Алекса даёт себе
- Моменты, когда маска slip

**Routine + psychological impact**:
- Технические действия описаны буднично
- Но показан накопительный эффект на Алексу

#### Примеры техник:

**Хорошо** ✅:
```
Третий клиент за день. Женщина. Средних лет. Травма изнасилования.

Алекса открыла файл. Профессионально. Смотрела на паттерн воспоминаний, не на лицо клиентки.

Удалить: ощущения. Страх. Беспомощность.
Оставить: факт. Знание. Необходимое для безопасности.

Чисто. Эффективно.

Клиентка вышла. Благодарила. Улыбалась.

Алекса выключила консоль. Руки дрожали. Незаметно. Профессионально.

Ещё двое до конца дня.
```

**Плохо** ❌:
```
Алекса редактировала воспоминания клиентки, чувствуя моральные сомнения относительно своей профессии, но продолжая работать, потому что это было её долгом.
```

#### Практические guidelines:

- Показывайте contrast: профессиональное действие vs внутреннее воздействие
- Routine described matter-of-factly
- Маленькие детали выдают психологическую цену: дрожь рук, потребность вымыть руки, усталость
- Алекса использует технический язык как защиту от эмоций
- Моменты empathy прорываются через маску - показывайте через действия (пауза, глубокий вдох)
- Никаких прямых моральных суждений - показ через накопительный эффект

### 8. Экспозиция мира

**Подход**: William Gibson
**Когда применять**: Описание вертикального города, социальной структуры, urban environment

#### Ключевые характеристики:

**Мир как фон**:
- Не centre stage, а контекст для действия
- Показ через использование и взаимодействие
- Никаких туристических описаний

**Детали через действие**:
- Персонаж движется по городу - мир раскрывается
- Специфичные детали вместо общих
- Вертикальная география естественно интегрирована

**Нормализация**:
- Для Алексы это обычный мир
- Странное для читателя обыденно для персонажа
- Show weirdness through casual mention

**Социальная структура через детали**:
- Уровни и классы показаны через: транспорт, одежду, качество воздуха, доступ
- Никаких прямых объяснений социальной системы

#### Примеры техник:

**Хорошо** ✅:
```
Глайдер снизился. Сто двадцатый этаж. Воздух здесь чище. Верхние уровни.

Через окно — огни нижних секторов. Далеко внизу. Алекса не спускалась туда три года. Не было причин.

Посадочная платформа — мрамор. Не пластик, как на шестьдесят третьем. Маркер уровня. Себастьян жил здесь.
```

**Плохо** ❌:
```
Вертикальный мегаполис был разделён на уровни, где богатые жили на верхних этажах с чистым воздухом, а бедные - внизу. Это была несправедливая система, отражающая социальное неравенство будущего.
```

#### Практические guidelines:

- Одна специфичная деталь вместо общего описания
- Вертикальная география через числа уровней casualy mentioned
- Социальное неравенство через material details (мрамор vs пластик)
- Технологии упоминаются как часть повседневности
- Никаких "In this world..." - просто показ
- World-building через контраст (верхние vs нижние уровни)
- Алекса не восхищается и не ужасается - просто живёт

### 9. Эмоциональные/личные моменты

**Референс**: Казуо Исигуро
**Когда применять**: Связь с котом, воспоминания о Реджинальде, моменты уязвимости, личные relationships

#### Ключевые характеристики:

**Сдержанность**:
- Эмоции не названы напрямую
- Understated prose
- Больше показано, чем выражено

**Эмоции через маленькие жесты**:
- Физические детали вместо эмоциональных деклараций
- Символические объекты (кот, фото, браслет)
- Повседневные ритуалы с emotional weight

**Элегантная простота**:
- Простые слова, но carefully chosen
- Rhythm спокойный, медитативный
- Avoid melodrama

**Недосказанность**:
- Алекса не формулирует чувства полностью
- Читатель понимает больше, чем персонаж говорит
- Vulnerability через маленькие моменты, не grand declarations

#### Примеры техник:

**Хорошо** ✅:
```
Кот свернулся на коленях. Тёплый. Мурлыканье вибрировало в тишине.

Алекса провела рукой по серебристой шерсти. Медленно. Как тогда. В ложном воспоминании.

Но кот настоящий. Она это знала. Хотела знать.

Мурлыканье не прекращалось.
```

**Плохо** ❌:
```
Алекса почувствовала глубокую связь с котом. Он был единственным доказательством, что её воспоминания о Реджинальде были реальными, и это наполняло её сердце теплом и надеждой.
```

#### Практические guidelines:

- Never name emotions directly ("она чувствовала любовь/грусть/тоску")
- Show through: touch, rhythm of breathing, small rituals
- Symbolic objects carry emotional weight
- Repetition of small actions (погладить кота, посмотреть на браслет)
- Простой язык, но evocative
- Тишина и паузы значимы
- Vulnerability через маленькие моменты weakness, не dramatic scenes
- Алекса позволяет себе feeling только наедине

## Комбинирование стилей

Многие сцены содержат элементы нескольких типов. В этом случае:

1. **Определите доминирующий тип** (что главное в сцене?)
2. **Используйте его стиль как основу**
3. **Добавьте элементы второстепенных типов**

### Пример: Социальное взаимодействие + Саспенс

Основа: Катон (формальность, подтекст, классовое напряжение)
Добавить: Thriller элементы (короткие абзацы в критические моменты, физиология стресса)

```
— Интересный проект, Алекса. — Себастьян листал её запросы. Медленно.

Формальный тон. Дистанция. Но он знал. Она видела это.

Пауза. Долгая. Професиональная маска держалась. Едва.

Пульс участился. Ладони влажные. Не показывать.

— Аттестационная подготовка, — сказала она. Ровно.

Он посмотрел. Долгий взгляд. Оценивающий.
```

### Пример: Профессиональная работа + Эмоциональный момент

Основа: Психологический (routine + internal impact)
Добавить: Исигуро (сдержанность, small gesture of vulnerability)

```
Клиент. Мужчина. Потеря ребёнка.

Алекса открыла файл. Стандартная процедура. Убрать острую боль. Оставить память без агонии.

Паттерн развернулся. Детский смех в воспоминании.

Её рука замерла над консолью. Секунду. Две.

Профессионально. Она завершила редактирование. Чисто. Правильно.

Клиент ушёл. Благодарил.

Алекса закрыла глаза. Ненадолго.
```

## Общие принципы (применимы ко всем стилям)

Независимо от типа сцены, всегда соблюдайте:

### 1. Show Don't Tell
- Эмоции через действия/dialogue/физические реакции
- Мир через взаимодействие, не exposition
- Характер через поведение, не описания

### 2. POV Consistency
- Всё через восприятие Алексы
- Только то, что она может знать/видеть/чувствовать
- Другие персонажи через её interpretation

### 3. Narrative Continuity
- Временная последовательность
- Пространственная логика
- Character knowledge accumulates
- Информация не повторяется без причины

### 4. Specific Over Generic
- Конкретные детали вместо общих описаний
- "Чёрное стекло" вместо "современное здание"
- "6 лет 4 месяца 18 дней" вместо "мало времени"

### 5. Fresh Language
- Никаких клише
- Избегать заезженных метафор
- Каждое сравнение должно быть unique to this world

### 6. Sensory Grounding
- Orient reader с сенсорными деталями
- Use multiple senses (не только visual)
- Специфичные для sci-fi world: холод металла, озон, искусственное освещение

## Практическое применение

### Workflow для prose-writer:

1. **Прочитать blueprint сцены**
2. **Определить тип сцены** (или комбинацию)
3. **Обратиться к соответствующему разделу guide**
4. **Применить техники к каждому beat**
5. **Проверить соответствие стилю** при редактировании

### Checklist перед завершением сцены:

- [ ] Стиль соответствует типу сцены?
- [ ] Длина предложений appropriate для тона?
- [ ] Sensory details grounding читателя?
- [ ] POV консистентен?
- [ ] Никаких sci-fi клише?
- [ ] Show, not tell выдержан?
- [ ] Диалоги с подтекстом (если есть)?
- [ ] Технологии показаны через использование?

## Исключения и гибкость

Этот guide — инструмент, не смирительная рубашка.

**Когда отступить от guide**:
- Artistic choice улучшает сцену
- Характер персонажа требует отклонения
- Specific moment требует unique approach

**Но всегда спросите себя**: Это обоснованное отклонение или просто easier way?
