# Planning Workflow

!!! info "Полная документация"
    
    Подробная техническая документация доступна в [`.workflows/planning.md`](../../../.workflows/planning.md)

Интерактивный многоуровневый процесс планирования сюжета от стратегического видения акта до детальных scene blueprints.

## Обзор

Planning Workflow состоит из 5 фаз, каждая с специализированными агентами:

1. **Exploration** — анализ текущего состояния мира и персонажей
2. **Scenario Generation** — генерация 3-5 вариантов развития
3. **Path Planning** — разбивка на события и сцены
4. **Scene Detailing** — детализация каждой сцены
5. **Integration** — интеграция в сюжетные линии
